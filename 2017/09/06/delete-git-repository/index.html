删除GitHub的仓库

1. 登录GitHub, 找到要删除的仓库

2. 点击Settings, 进入仓库的设置界面

3. 点击Delete this repository 删除这个仓库

输入对应的仓库名, 点击确认即可删除(可能会要求输入GitHub的密码).
